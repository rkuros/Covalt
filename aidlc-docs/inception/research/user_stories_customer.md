# ユーザーストーリー: 顧客（エンドユーザー）

## EP-01: LINE連携基盤

### US-C01: LINE公式アカウントの友だち追加

**ストーリー**: 顧客として、LINE公式アカウントを友だち追加したい。それにより、予約システムを利用できるようになる。

**アクセプタンスクライテリア**:
- [ ] QRコードまたはリンクからLINE公式アカウントを友だち追加できる
- [ ] 友だち追加完了時にウェルカムメッセージが自動送信される
- [ ] ウェルカムメッセージにシステムの利用方法の簡単な案内が含まれる
- [ ] 友だち追加時に顧客情報がシステムに自動登録される（LINEユーザーID、表示名）

### US-C02: リッチメニューからの操作

**ストーリー**: 顧客として、LINEトーク画面のリッチメニューから主要な操作にアクセスしたい。それにより、迷わずに予約や確認ができる。

**アクセプタンスクライテリア**:
- [ ] トーク画面下部にリッチメニューが常時表示される
- [ ] リッチメニューから「予約する」をタップするとLIFFアプリ（予約画面）が起動する
- [ ] リッチメニューから「予約確認」をタップすると予約一覧が表示される
- [ ] リッチメニューから「予約履歴」をタップすると過去の予約履歴が表示される

### US-C03: LIFFアプリの起動と認証

**ストーリー**: 顧客として、LINEアプリ内で予約画面（LIFFアプリ）をスムーズに開きたい。それにより、別アプリへの切り替えなしに予約操作ができる。

**アクセプタンスクライテリア**:
- [ ] リッチメニューまたはメッセージ内のリンクからLIFFアプリが起動する
- [ ] LINEの認証情報が自動的に引き継がれ、追加ログインが不要である
- [ ] LIFFアプリ起動時にLINEユーザー情報（ユーザーID、表示名）が取得される

## EP-02: 予約管理（顧客側）

### US-C04: 空き状況の確認

**ストーリー**: 顧客として、希望する日時の空き状況を確認したい。それにより、予約可能な日時を把握した上で予約できる。

**アクセプタンスクライテリア**:
- [ ] LIFFアプリ上でカレンダーUIから日付を選択できる
- [ ] 選択した日付の空きスロット（時間帯）が一覧表示される
- [ ] 空きスロットは「予約可能」「予約済み」の状態が視覚的に区別できる
- [ ] 過去の日付は選択できない
- [ ] オーナーが設定した休業日は選択できない（または休業日であることが表示される）

### US-C05: 予約の新規作成

**ストーリー**: 顧客として、希望する日時で予約を作成したい。それにより、サービスの提供を受ける約束を確定できる。

**アクセプタンスクライテリア**:
- [ ] LIFFアプリ上で空きスロットを選択して予約を作成できる
- [ ] 予約作成前に確認画面が表示される（日時の確認）
- [ ] 予約作成が完了すると完了画面が表示される
- [ ] 予約作成完了後、LINEトーク画面に予約確定メッセージが送信される
- [ ] 同一時間帯に重複した予約はできない（二重予約防止）
- [ ] 予約作成時にシステムが一意の予約IDを発行する

### US-C06: 予約の変更

**ストーリー**: 顧客として、作成済みの予約の日時を変更したい。それにより、予定が変わった場合にキャンセルせずに対応できる。

**アクセプタンスクライテリア**:
- [ ] 自分の予約一覧から変更したい予約を選択できる
- [ ] 変更先の空きスロットを選択できる
- [ ] 変更前に確認画面が表示される（変更前後の日時の比較）
- [ ] 変更が完了すると、LINEトーク画面に変更完了メッセージが送信される
- [ ] 変更先の時間帯が既に予約済みの場合は変更できない
- [ ] 過去の予約は変更できない

### US-C07: 予約のキャンセル

**ストーリー**: 顧客として、作成済みの予約をキャンセルしたい。それにより、行けなくなった場合に事前に取り消しができる。

**アクセプタンスクライテリア**:
- [ ] 自分の予約一覧からキャンセルしたい予約を選択できる
- [ ] キャンセル前に確認画面が表示される
- [ ] キャンセルが完了すると、LINEトーク画面にキャンセル完了メッセージが送信される
- [ ] キャンセルされたスロットは再び「予約可能」状態に戻る
- [ ] 過去の予約はキャンセルできない

### US-C08: 予約一覧の確認

**ストーリー**: 顧客として、自分の今後の予約一覧を確認したい。それにより、予約状況をいつでも把握できる。

**アクセプタンスクライテリア**:
- [ ] リッチメニューまたはLIFFアプリから今後の予約一覧を表示できる
- [ ] 各予約の日時・ステータス（確定・キャンセル済み等）が表示される
- [ ] 予約一覧から個別の予約の詳細を確認できる
- [ ] 予約一覧から変更・キャンセル操作に遷移できる

### US-C09: 予約履歴の確認

**ストーリー**: 顧客として、過去の予約履歴を確認したい。それにより、これまでの利用状況を振り返ることができる。

**アクセプタンスクライテリア**:
- [ ] 過去の予約（完了済み・キャンセル済み）を一覧で確認できる
- [ ] 各予約の日時・ステータスが表示される
- [ ] 直近の履歴から順に表示される

## EP-06: 通知（顧客向け）

### US-C10: 予約確定通知の受信

**ストーリー**: 顧客として、予約が確定したらLINEで通知を受け取りたい。それにより、予約が正しく完了したことを確認できる。

**アクセプタンスクライテリア**:
- [ ] 予約作成完了時にLINEトークに確定通知メッセージが届く
- [ ] 通知メッセージに予約日時が記載されている
- [ ] 通知メッセージに予約IDが記載されている

### US-C11: 予約変更通知の受信

**ストーリー**: 顧客として、予約が変更されたらLINEで通知を受け取りたい。それにより、変更内容を確認できる。

**アクセプタンスクライテリア**:
- [ ] 予約変更完了時にLINEトークに変更通知メッセージが届く
- [ ] 通知メッセージに変更前後の日時が記載されている
- [ ] オーナー側から予約が変更された場合にも通知が届く

### US-C12: 予約キャンセル通知の受信

**ストーリー**: 顧客として、予約がキャンセルされたらLINEで通知を受け取りたい。それにより、キャンセルが正しく処理されたことを確認できる。

**アクセプタンスクライテリア**:
- [ ] 予約キャンセル完了時にLINEトークにキャンセル通知メッセージが届く
- [ ] 通知メッセージにキャンセルされた予約の日時が記載されている
- [ ] オーナー側から予約がキャンセルされた場合にも通知が届く

### US-C13: 予約リマインダー通知の受信

**ストーリー**: 顧客として、予約日の前にリマインダー通知を受け取りたい。それにより、予約を忘れずに来店できる。

**アクセプタンスクライテリア**:
- [ ] 予約日の前日にLINEトークにリマインダーメッセージが届く
- [ ] リマインダーメッセージに予約日時が記載されている
- [ ] キャンセル済みの予約にはリマインダーが送信されない
