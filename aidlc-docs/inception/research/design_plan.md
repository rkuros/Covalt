# Design Plan Doc

## 1. 概要

個人事業主向けLINE予約システムのUI設計方針を定義する。
本ドキュメントは `mock.html` の画面モックに基づく。

| 項目 | 内容 |
|---|---|
| システム名 | 個人事業主向けLINE予約システム |
| LINE連携方式 | 方式C（Messaging API + LIFF） |
| インフラ構成 | AWS（CloudFront + S3 + API Gateway + Lambda） |
| モックファイル | [mock.html](./mock.html) |

---

## 2. アクターとデバイス

| アクター | 利用デバイス | インターフェース |
|---|---|---|
| 顧客（エンドユーザー） | スマートフォン | LINEアプリ内（トーク画面 + LIFFアプリ） |
| 個人事業主（オーナー） | タブレット（iPad）/ スマートフォン | Webブラウザ管理画面 |
| システム管理者 | PC / タブレット | Webブラウザ管理画面（オーナー画面の拡張） |

---

## 3. 顧客側画面設計（LINE / LIFFアプリ）

### 3.1 画面構成

```
LINEトーク画面
├── チャットエリア（通知メッセージ表示）
└── リッチメニュー（常時表示・3ボタン）
     ├── 「予約する」  → LIFFアプリ: 予約作成フロー
     ├── 「予約確認」  → LIFFアプリ: 予約一覧
     └── 「予約履歴」  → LIFFアプリ: 過去の予約一覧
```

### 3.2 画面一覧

| 画面ID | 画面名 | 対応ユーザーストーリー | 説明 |
|---|---|---|---|
| C-01 | LINEトーク画面 | US-C01, C02 | ウェルカムメッセージ、通知表示、リッチメニュー |
| C-02 | 日時選択画面 | US-C04 | カレンダーUI + 空きスロット一覧 |
| C-03 | 予約確認画面 | US-C05 | 日時・予約IDの確認、確定ボタン |
| C-04 | 予約完了画面 | US-C05 | 完了メッセージ、閉じるボタン |
| C-05 | 予約一覧画面 | US-C08 | 今後の予約リスト、各カードタップで詳細へ |
| C-06 | 予約詳細画面 | US-C08 | 予約ID・日時・ステータス、変更/キャンセルボタン |
| C-07 | 日時変更画面 | US-C06 | カレンダーUI + 空きスロット選択 |
| C-08 | キャンセル確認ダイアログ | US-C07 | モーダル確認 |
| C-09 | 予約履歴画面 | US-C09 | 過去の予約リスト（完了・キャンセル済） |

### 3.3 画面遷移

```
リッチメニュー「予約する」
  → C-02 日時選択（カレンダー → スロット選択）
    → C-03 予約確認
      → C-04 予約完了
      → トーク画面に確定通知メッセージ (US-C10)

リッチメニュー「予約確認」
  → C-05 予約一覧
    → C-06 予約詳細
      → C-07 日時変更 → スロット選択で変更確定 → トーク画面に変更通知 (US-C11)
      → C-08 キャンセル確認 → 確定 → トーク画面にキャンセル通知 (US-C12)

リッチメニュー「予約履歴」
  → C-09 予約履歴
```

### 3.4 通知メッセージ（チャットエリア）

| 通知種別 | トリガー | 表示内容 | 対応US |
|---|---|---|---|
| 予約確定 | 予約作成完了時 | 日時、予約ID | US-C10 |
| 予約変更 | 予約変更完了時 | 変更前後の日時、予約ID | US-C11 |
| キャンセル | キャンセル完了時 | 日時、予約ID | US-C12 |
| リマインダー | 予約前日（バッチ） | 日時 | US-C13 |

### 3.5 カレンダーUI仕様

- 月表示、左右矢印で月切替
- 過去日付: グレーアウト、選択不可
- 休業日: 赤文字+取消線、選択不可
- 今日: 緑太字ハイライト
- 選択日: 緑背景白文字
- 日付選択後にその日の空きスロット一覧を表示

## 4. オーナー側画面設計（Web管理画面）

### 4.1 画面構成

```
タブレット画面
├── トップバー（システム名、ログアウト）
├── コンテンツエリア（スクロール可能）
└── ボトムナビゲーション（4タブ）
     ├── 予約一覧
     ├── スケジュール
     ├── 顧客
     └── 設定
```

### 4.2 画面一覧

| 画面ID | 画面名 | タブ | 対応ユーザーストーリー | 説明 |
|---|---|---|---|---|
| O-00 | ログイン画面 | - | US-S01, S02 | メール/パスワード認証、パスワードリセットリンク |
| O-01 | 予約一覧（リスト） | 予約一覧 | US-O01 | 日/週/月/カレンダー切替、テーブル形式 |
| O-02 | 予約一覧（カレンダー） | 予約一覧 | US-O01 | 月間カレンダーグリッド、日付セルに予約表示 |
| O-03 | 予約詳細 | 予約一覧 | US-O02 | 予約ID・日時・顧客名・ステータス |
| O-04 | 手動予約作成ダイアログ | 予約一覧 | US-O03 | 顧客選択、日付、時間指定 |
| O-05 | 予約変更ダイアログ | 予約一覧 | US-O04 | 時間帯変更 |
| O-06 | 予約キャンセル確認ダイアログ | 予約一覧 | US-O05 | モーダル確認 |
| O-10 | 営業時間設定 | スケジュール | US-O06 | 曜日ごとの営業/定休、開始/終了時間 |
| O-11 | 休業日設定 | スケジュール | US-O07 | 日付入力で追加、タグ形式で一覧表示 |
| O-12 | 空きスロット管理 | スケジュール | US-O08 | グリッド形式、予約あり/なし表示 |
| O-20 | 顧客一覧 | 顧客 | US-O09 | テーブル形式、名前検索、LINE連携状態 |
| O-21 | 顧客詳細 | 顧客 | US-O10, O11 | 顧客情報編集、予約履歴表示 |
| O-22 | 新規顧客登録ダイアログ | 顧客 | US-O11 | 顧客名入力 |
| O-30 | LINE接続設定 | 設定 | US-S05 | トークン、シークレット、LIFF ID、Webhook URL |
| O-31 | Googleカレンダー連携 | 設定 | US-O14, O15 | Google認証連携ボタン |

### 4.3 画面遷移

```
O-00 ログイン
  → 認証成功 → O-01 予約一覧（リスト/週表示がデフォルト）

予約一覧タブ
  O-01 リスト表示 ←→ O-02 カレンダー表示（トグル切替）
  O-01/O-02 → 予約タップ → O-03 予約詳細
    → O-05 変更ダイアログ → 確定 → O-03に戻る + LINE通知送信
    → O-06 キャンセルダイアログ → 確定 → O-01に戻る + LINE通知送信
  O-01 → 「+ 手動予約」 → O-04 手動予約作成ダイアログ → 確定 → O-01に戻る

スケジュールタブ
  O-10 営業時間設定
  O-11 休業日設定
  O-12 空きスロット管理
  ※ 1ページ内に縦スクロールで全セクション表示

顧客タブ
  O-20 顧客一覧 → 顧客タップ → O-21 顧客詳細 → 「一覧に戻る」 → O-20
  O-20 → 「+ 新規登録」 → O-22 新規顧客登録ダイアログ → 確定 → O-20

設定タブ
  O-30 LINE接続設定（接続テストボタン）
  O-31 Googleカレンダー連携
  ※ 1ページ内に縦スクロールで全セクション表示
```

### 4.4 カレンダービュー仕様（オーナー側）

- 月間グリッド表示、左右矢印で月切替
- 各日付セルに該当日の予約を色付きバーで表示
  - 確定: 緑（`#e8f5e9`）
  - キャンセル済: 赤（`#fce4ec`）
  - 完了: 青（`#e3f2fd`）
- 今日の日付セル: 青背景ハイライト
- 休業日セル: 赤背景、日付数字に取消線
- 予約バーをタップすると予約詳細画面へ遷移
- 前月/翌月の日付はグレーアウト表示

## 5. 共通UIパターン

### 5.1 インタラクション

| パターン | 用途 | 実装 |
|---|---|---|
| モーダルダイアログ | キャンセル確認、手動予約作成、日時変更 | 背景オーバーレイ付きセンター表示 |
| トースト通知 | 操作完了フィードバック | 右上に2.5秒表示後フェードアウト |
| LIFFオーバーレイ | 顧客側の各機能画面 | LINE画面上に全画面オーバーレイ |

### 5.2 ステータス表示

| ステータス | 色 | バッジ背景 |
|---|---|---|
| 確定 | 緑 `#2e7d32` | `#e8f5e9` |
| キャンセル済 | 赤 `#c62828` | `#fce4ec` |
| 完了 | 青 `#1565c0` | `#e3f2fd` |

### 5.3 カラーパレット

| 用途 | 色 |
|---|---|
| LINE（顧客側プライマリ） | `#06c755` |
| 管理画面プライマリ | `#0984e3` |
| トップバー/ダーク | `#2d3436` |
| 背景 | `#f5f6fa` |
| 危険操作 | `#d32f2f` |
| テキスト | `#333333` |
| サブテキスト | `#999999` |
