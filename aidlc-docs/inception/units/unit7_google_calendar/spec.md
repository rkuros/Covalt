# Unit 7: Googleカレンダー連携

## 境界づけられたコンテキスト

オーナーのGoogleカレンダーと予約情報を同期するコンテキスト。Google OAuth認証、カレンダー選択、予約イベントの自動同期を担う。

## ユーザーストーリー

### US-O14: Googleカレンダー連携の設定【Could】

**ストーリー**: オーナーとして、自分のGoogleカレンダーとシステムを連携したい。それにより、普段使っているカレンダーで予約状況を確認できる。

**アクセプタンスクライテリア**:
- [ ] Web管理画面からGoogleアカウントで認証・連携設定ができる
- [ ] 連携するカレンダーを選択できる
- [ ] 連携を解除できる

---

### US-O15: 予約のGoogleカレンダー自動同期【Could】

**ストーリー**: オーナーとして、予約が作成・変更・キャンセルされた際にGoogleカレンダーに自動反映されてほしい。それにより、手動で転記する手間が省ける。

**アクセプタンスクライテリア**:
- [ ] 予約が作成されると、Googleカレンダーに予定が自動追加される
- [ ] 予約が変更されると、Googleカレンダーの予定が自動更新される
- [ ] 予約がキャンセルされると、Googleカレンダーの予定が自動削除される
- [ ] Googleカレンダーの予定に予約日時・顧客名が記載される
