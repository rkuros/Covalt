# Unit 6: 顧客情報管理

## 境界づけられたコンテキスト

オーナーが顧客情報を管理するコンテキスト。Web管理画面での顧客一覧・詳細・編集、およびLINE友だち追加時の自動登録を担う。

## ユーザーストーリー

### US-O09: 顧客一覧の表示【Could】

**ストーリー**: オーナーとして、顧客の一覧を確認したい。それにより、全体の顧客状況を把握できる。

**アクセプタンスクライテリア**:
- [ ] Web管理画面で顧客一覧が表示される
- [ ] 顧客名で検索できる
- [ ] LINE連携済みかどうかが表示される

---

### US-O10: 顧客詳細の確認【Could】

**ストーリー**: オーナーとして、個別の顧客の詳細情報と予約履歴を確認したい。それにより、顧客に合わせた対応ができる。

**アクセプタンスクライテリア**:
- [ ] 顧客一覧から個別の顧客をクリックして詳細画面を開ける
- [ ] 顧客名・LINE表示名・登録日が表示される
- [ ] その顧客の予約履歴（過去・今後）が表示される

---

### US-O11: 顧客情報の編集【Could】

**ストーリー**: オーナーとして、顧客の情報を編集したい。それにより、顧客情報を最新の状態に保てる。

**アクセプタンスクライテリア**:
- [ ] 顧客詳細画面から顧客名等の情報を編集できる
- [ ] 編集内容を保存できる
- [ ] 手動で新規顧客を登録できる（LINE連携なしの顧客にも対応）
